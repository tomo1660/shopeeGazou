# Shopee ストアデコ作成ツール

Shopeeストアのデコレーション画像(1080×1080px)を国別に最大15枚作成・ダウンロードできるNext.jsアプリです。

---

## セットアップ (初回のみ)

### 1. Node.js をインストール

https://nodejs.org/ja/ から **LTS版** をダウンロードしてインストール。

インストール後、PowerShellを**再起動**して以下で確認:
```
node --version   # v18以上
npm --version    # v9以上
```

### 2. 依存パッケージをインストール

```powershell
cd "C:\Users\Owner\Desktop\claudeコード保存\shopee-store-deco"
npm install
```
（初回は3〜5分かかります）

### 3. アプリを起動

```powershell
npm run dev
```

ブラウザで → **http://localhost:3000**

## 使い方

### 基本フロー

1. **国を選択** — ページ上部のタブ (SG/PH/MY/TH/TW/VN/BR)
2. **フレーム設定** (任意) — 左パネル「フレーム管理」でフレーム画像をアップロード。1枚目専用で、全面背景として貼り付け
3. **画像を追加** — 「ファイルから選択」 または キャンバスをクリックして `Ctrl+V` でクリップボードから貼付
4. **テキスト追加** — 右パネル「テキスト」で文字入力→「キャンバスに追加」。追加後にダブルクリックで直接編集可能
5. **翻訳** — 右パネル「翻訳」でテキスト翻訳 or 画像OCR翻訳。結果を「画像に追加」で貼付
6. **スロット切り替え** — 画面下部のサムネイルで最大15枚を管理
7. **ダウンロード** — 「全画像をZIPでダウンロード」ボタンで1080×1080px PNG一括保存

### キャンバス操作

| 操作 | 説明 |
|------|------|
| クリック | キャンバスをアクティブ化（編集モード） |
| Ctrl+V | 画像または翻訳テキストを貼付 |
| Del / Backspace | 選択オブジェクトを削除 |
| ドラッグ | オブジェクトを移動 |
| コーナーハンドル | オブジェクトをリサイズ・回転 |
| ダブルクリック | テキストを直接編集 |

## 機能一覧

### 対応国・言語
| 国 | 言語 |
|----|------|
| 🇸🇬 シンガポール | English |
| 🇵🇭 フィリピン | Filipino |
| 🇲🇾 マレーシア | Melayu |

| 🇹🇼 台湾 | 繁體中文 |
| 🇻🇳 ベトナム | Tiếng Việt |
| 🇧🇷 ブラジル | Português |

### フレーム
- 各国につき最大3枚のフレーム画像を保存
- フレームはキャンバス全体に引き伸ばして背景として表示
- 1枚目のみフレーム適用可能（2〜15枚目はなし）

### テキスト
- フォント: Noto Sans JP / Arial / Georgia / Impact など9種
- サイズ: 8〜200px
- 色・太字・斜体・下線・縁取りを設定
- 左/中央/右揃え

### 翻訳（無料API使用）
- **テキスト翻訳**: 11言語対応（MyMemory無料APIを使用）
- **画像OCR翻訳**: Tesseract.jsで画像からテキスト抽出→翻訳
  ※初回OCR実行時に言語データをダウンロードします（数秒かかります）

### ダウンロード
- 全スロットをオフスクリーンレンダリングして1080×1080px PNGで出力
- ZIP形式で一括ダウンロード（`shopee-[国コード].zip`）

---

## 注意事項

- 画面上のキャンバスは540×540pxで表示（ダウンロード時に2倍の1080×1080pxで出力）
- ページを閉じるとデータは消えます（ローカル保存機能なし）
- 翻訳APIは無料のため、1日の利用制限あり（超過するとエラー表示）
